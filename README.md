# 🤖 微信防剧透机器人 v1.0
## 使用
### 安装

- 项目基于Python3
- 安装 (如存在多Python共存问题，选择第二个命令):

    `pip install fuck-spoiler`

    `pip3 install fuck-spoiler`
### 卸载

`pip unstall fuck-spoiler` 
    
`pip3 unstall fuck-spoiler` 
    
### 使用

1. 运行`fuck name`，name为你想防止被剧透的剧名

   e.g: `fuck 蜘蛛侠`
    
2. 输入之后等待几秒，会跳出二维码，扫描登录即可开启防剧透

### 退出
按`control` + `c`，即可退出
